Fluid OSK Examples

These examples demonstrate two key use cases for this library.

Onscreen Keyboard, HTML Text Input

The HTML text input demo illustrates how the keyboard can be configured to send text to a form element.

Onscreen Keyboard, Custom Input

The custom input demo illustrates how the keyboard can be configured to send text to a form element.

Typing Feedback

The typing feedback demo shows how to represent the state of a keyboard without providing clickable and navigable buttons.
